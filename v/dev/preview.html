Introduction


This Developer Preview is currently in Pre-Alpha Testing stage and for raw testing only.
When entering the world, you admit that this project is in no way near finished, nor is it playable with any kind of multi-user experiences.

This Pre-Alpha Testing is solely for educational purposes and to test the raw implemented functions. This Pre-Alpha Testing is meant to find bugs in it's current early state and not meant to entertain anyone yet. This Pre-Alpha Testing is mainly to test the server performance and it's stability with web rendering in multiple web browsers. This Pre-Alpha Testing is to get first opinions, suggestions and expectations by a first early viewer base. This Pre-Alpha Testing stage can be closed at any time without previous notification.

Functions


This Developer Preview includes the following functions, but is not limited to or can and will be changed in any possible time.

Movement:
W A S D for walking in each direction
Move your mouse to walk to it's direction
Hold SHIFT while walking to walk faster

Camera:
Use Scroll-Wheel to zoom more in and out
Hold CTRL and move your mouse to inspect your character

Animation:
Walking with W has an own walk animation
Running with W and SHIFT has it's own run animation
Number-Keys 1, 2, 3 and 4 are set to trigger NPC animations
